# Emesis-Crosshairs
Custom TF2 crosshair pack with hitscan and projectile styles.

## Features
-  Hitscan crosshair (green dot)
-  Projectile crosshair (red circle)
-  Weapon slot binds for automatic switching
-  Easy to install and customize

## Installation
1. Download `steeledge_crosshairs.cfg`.
2. Place it in your TF2 `tf/cfg` folder:
'SteamLibrary/steamapps/common/Team Fortress 2/tf/cfg'
 Launch TF2 and run in console: exec steeledge_crosshairs
--
## Optional Auto-Load
Add this line to your `autoexec.cfg`:
exec steeledge_crosshairs
